# Communication System Visualizer

A **Streamlit-based interactive app** for visualizing digital communication systems.

This project focuses on:

- Signal generation
- Modulation techniques
- Channel effects
- Noise modeling
- Visualization of communication blocks

Built for **education, experimentation, and understanding core communication system concepts**.

---

## Features

- Interactive signal visualizations
- Digital modulation schemes
- Channel simulation
- Real-time plots
- Educational UI

---

## Tech Stack

- Python
- Streamlit
- NumPy
- Scipy
- Matplotlib / Plotly
  n---

## Run Locally

```bash
pip install -r requirements.txt
pre-commit install
streamlit run app.py
```
